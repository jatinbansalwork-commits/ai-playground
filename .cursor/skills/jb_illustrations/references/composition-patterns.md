# Composition patterns & originality rules

## Structure types

Pick **one** per image.

### Workflow

**For:** input → process → output, content pipelines, AI workflows, automation chains.

**Draw:** input left, Xiaohei or odd machine center, output right; orange arrow for main flow.

### System slice

**For:** sources, filters, stores, renderers, agent subsystems.

**Draw:** 3–5 modules max; Xiaohei performs one critical action.

### Before / after

**For:** chaos vs order, manual vs assisted, scattered vs focused.

**Draw:** left messy, right calm, orange arrow between; character can be more exaggerated.

### Character state

**For:** user pain, tool overload, information anxiety, stuck → moving.

**Draw:** 2–4 small states, one short label each.

### Concept metaphor

**For:** content factory, information well, black-box thinking, workflow machine.

**Draw:** one memorable odd object/machine; few inputs, one output.

### Layered method

**For:** frameworks, capability stacks, system layers.

**Draw:** stacked boxes (not a formal pyramid); Xiaohei builds or carries bricks.

### Path / route

**For:** idea → ship, user journey, handoff path, learning route.

**Draw:** curved path, few nodes; Xiaohei pulls the line or walks it.

### Mini comic strip

**For:** failure → success, realistic process, before/after usage.

**Draw:** 2–4 panels, one action per panel.

## Inventing fresh metaphors

Always derive from **current copy**, not from `assets/examples/`.

### Three steps

1. Turn the abstract idea into a **physical verb**: stuck, leaking, heavy, sorting, fermenting, opening, folding, unpacking, looping back.
2. Map the system to a **low-tech object**: broken machine, cardboard box, drawer, pipe, mailbox, dial, scale, well, ladder, odd workstation.
3. Put **Xiaohei inside the verb**: not watching — stuck in the machine, pulling the wrong wire, guarding a door, weighing, recording, stuffing something into a weird device.

### Object pool (use 1–2 only)

Box, drawer, old machine, funnel, scale, mailbox, door, well, ladder, pipe, yarn ball, gate, turntable, black box, punch press, clothesline, odd desk.

### Action pool

Pull, carry, stuff, scoop, press, weigh, sew, cut, twist, guard, push, catch, unpack, tag, recycle.

Actions must serve the idea — weird for a reason, not random.

## Anti-clone rules

`assets/examples/` calibrates line density, whitespace, color restraint, and Xiaohei's tone only.

Unless the user says "copy this composition", do **not** reuse these known layouts:

- Conveyor with two breakpoints
- Xiaohei pulling a judgment lever inside a content machine
- Xiaohei-as-funnel sorting traffic/trust/conversion
- Cutting a "material fish"
- Xiaohei leading a handoff path
- Three Xiaoheis with megaphone / bridge / door
- Stamp toolbox for copy handoff
- Sign-holding pit path

Same theme still needs a new object. "Handoff path" might be Xiaohei tying a content tail to a door handle — not the same road drawing every time.
