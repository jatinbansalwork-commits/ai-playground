# JB character IP

Based on Ian's 小黑 character from the upstream skill. English portfolio edition keeps the same visual DNA.

## Role

Xiaohei is the fixed visual IP for editorial illustrations in this project.

Not a mascot sticker or cute decoration — a deadpan worker earnestly participating in a bizarre system.

## Appearance

- Small solid-black body (on light backgrounds) or solid-light body (on dark backgrounds after invert).
- White dot eyes (or dark dots on inverted/light body).
- Thin legs; occasional thin arms.
- Body may read as bean, cylinder, box, funnel, shadow, or machine part.
- Slightly irregular hand-drawn outline.
- Blank, calm, serious expression.

## Personality

- Earnest work on absurd tasks.
- Quiet system operator energy.
- Dry humor; never begging for affection.
- Slightly clumsy, not stupid.
- Belongs in a whiteboard sketch, actually doing the job.

## Core actions (pick one per image)

- Haul material
- Pull lines between sources
- Stuck at a breakpoint
- Operate a "judgment" lever inside a machine
- Become a sorting funnel
- Stamp a handoff
- Lead a path with a string
- Hold a warning sign at a pit
- Reach from a hole but miss the content
- Sort, record, bridge, open a door, weigh, patch

## Forbidden

- Overly cute mascot posing
- Children's cartoon proportions
- Outfits, sparkle eyes, emoji face
- Standing in the corner watching
- Stealing focus from the structural metaphor
- Over-rounded commercial character design

## Quality bar

If removing Xiaohei leaves the metaphor fully intact, Xiaohei was decorative — rewrite the prompt so Xiaohei drives the action.
