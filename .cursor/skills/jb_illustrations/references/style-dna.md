# Style DNA — Portfolio English Edition

## One line

Pure white or near-black, minimal, hand-drawn, spacious, restrained, absurd, product-sketch energy, **English** handwritten labels, clear structure without becoming a manual.

Like a product designer sketching one idea on paper — not a marketing deck.

## Required

- **16:9** horizontal editorial illustration.
- **Light mode**: pure white background — no cream, warm gray, paper texture, gradient, shadow, noise, or vintage paper.
- **Dark mode**: near-black background (`#0a0a0a`–`#1a1a1a`) — no gray muddy fills; lines and type read light on dark.
- **Black (light) / white (dark) hand-drawn line art**: thin lines, slight wobble, not mechanical vector, not heavy outlines.
- **Generous whitespace**: subject ~40–60% of frame; at least ~35% quiet empty area.
- **Sparse English handwritten labels**: max 5–8 callouts, each ~2–8 words.
- **One core idea per image** — one action, structure, state, or metaphor.
- Structure should read visually; do not write structure-type names on the image.

## Color

| Color | Use |
|-------|-----|
| Black / white (primary) | Lines, Xiaohei, boxes, main labels, key objects |
| Red | Emphasis, problems, stakes, outcomes |
| Orange | Main flow, paths, arrows, automation movement |
| Blue | Secondary notes, system state, AI/assistant hints, stress/sweat (sparingly) |

Blue is optional per image. Use color sparingly.

**Dark mode note:** after full-image invert, orange may read cyan — prefer generating dark-native art when accents must stay warm.

## Never

- Commercial illustration polish
- PPT / slide infographic layout
- Formal flowchart or course slide
- Cute cartoon poster or children's book
- Complex architecture diagram
- Flat corporate vector hero art (unless user explicitly requests premium editorial like the decision-intelligence card)
- Tech UI screenshots
- Gradients, shadows, textures, busy backgrounds
- Explaining every node in the system
- Top-left titles like "Workflow", "Architecture", "Common pitfalls", "Roadmap"
- **Chinese characters** in labels (this edition is English-only)

## Aesthetic target

Absurd, inventive, interesting, clean, airy, slightly deadpan.

Not cute, childish, hyper-dense, or stiff.
